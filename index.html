NEDEN MEMO OS?

Mehmet Can YILDIRIM tarafından kodlanan bu sistem, Linux çekirdeğinin gücüyle Frutiger Aero'nun cam estetiğini birleştirir. Dijital bağımsızlığın en modern temsilcisidir.

📦 APT Paket Yönetimi

Güvenli sunuculardan en yeni yazılımları indirin ve kurun.

sudo apt-get update && upgrade
# Memo OS Paket Deposu

🛡️ IP Tables Güvenliği

Gelişmiş koruma duvarını kernel seviyesinde yönetin.

sudo iptables -A INPUT -p tcp --dport 80 -j ACCEPT

🧩 Systemd Servisleri

Aero arayüz servislerini systemd ile optimize edin.

sudo systemctl enable memo-aero.service
